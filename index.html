Nosso Hardware Arduino

Desenvolvemos uma solução personalizada baseada no Arduino Uno, com componentes adicionais para maior funcionalidade e desempenho.

Placa Principal

Arduino Uno R3 com shield personalizado

Conectividade

Speaker(caixinha de som), Cabo USB, Smartphone

Sensores

Modulo Sensor De Gas Mq-2

Código Principal

// Projeto Arduino - Controle Principal
#define entradaDigital 0
#define entradaAnalogica 0

bool dSensor;
int aSensor;
int buzzerPin = 7;
void setup() {
  Serial.begin(9600);

  pinMode (entradaDigital, INPUT);

}

void loop() {
  dSensor = digitalRead(entradaDigital);
aSensor = analogRead(entradaAnalogica);

Serial.print("{\"dSensor\":");
Serial.print(dSensor);

Serial.print(",\"aSensor\":");
Serial.print(aSensor);
Serial.println("}");

delay(1000);

if(aSensor >= 250){


 tone(7, aSensor);
}
else{
   noTone(buzzerPin);
}
}
}

Especificações Técnicas

  • Processador: ATmega328P
  • Memória: 32KB Flash, 2KB SRAM
  • Conectividade: Smartphone + USB Tipo B
  • Portas: 14 digitais, 6 analógicas
  • Alimentação: 5V DC ou USB
